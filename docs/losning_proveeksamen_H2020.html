import math

def exp_approx(x,n):
    taylor_exp = 0
    for k in range(0,n+1):
        taylor_exp += x**k/math.factorial(k)
    return taylor_exp

Oppgave 12¶

Lister er datastrukturer i Python som kan tilordnes en samling variabler av samme eller ulik type. Når vi adderer lister til hverandre, legges listene etter hverandre (som når vi bruker append). Arrayer er datastrukturer som tillater flere matematiske operasjoner. Dersom vi for eksempel ganger eller adderer to arrayer, får vi komponentvis multiplikasjon og addisjon av arrayene:

array1 = [1, 2, 3]

array2 = [2, 2, 3]

array3 = array1*array2 = [1*2, 2*2, 3*3] = [2, 4, 9]

Vi kan ikke utføre komponentvis matematiske operasjoner med lister. Siden arrayer kan manipuleres komponentvis, er det nyttig når vi skal f.eks. plotte funksjoner:

x = np.linspace(-100, 100, 10000)

y = x**2 - 2

eller

def f(x): return x**2 - 2

y = f(x)

Oppgave 14¶

a)¶

I Newtons metode ønsker vi å bruke nullpunktet til tangenten som en tilnærming til nullpunktet til funksjonen. Det vil si at:

\[y = 0\]
\[f(x_0) + f'(x_0)(x-x_0) = 0\]

Siden vi ønsker å finne et uttryk for hvilken x-verdi som gir \(y = 0\), må vi omforme uttrykket:

\[f'(x_0)(x-x_0) = - f(x_0) \]
\[x-x_0 = - \frac{f(x_0)}{f'(x_0)}\]
\[ x = x_0 - \frac{f(x_0)}{f'(x_0)}\]

som er Newtons metode. Vi ønsker å gjenta denne metoden, så vi formulerer gjerne metoden iterativt som en differenslikning:

\[ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\]

b)¶

Vi kan bruke metoden til å finne nullpunkter til funksjoner. Siden likninger også kan skrives som venstreside \(–\) høyreside = 0, kan vi også løse likninger med denne metoden. I kjemi kan metoden for eksempel brukes til å finne ut ved hvilken tid konsentrasjonen til to stoffer er lik gitt to funksjonsuttrykk som beskriver konsentrasjonen som funksjon av tid, \(c_1(t)\) og \(c_2(t)\). Da kan vi løse likningen \(c_1(t) - c_2(t) = 0\) ned Newtons metode. Dette gjelder også dersom funksjonene er analytisk uløselige.

c)¶

Vi bruker en toleranseparameter fordi numeriske metoder som Newtons metode sjeldent gir eksakte svar. Vi bør derfor ikke sjekke om f(x) = 0, men heller sjekke om f(x) ligger nærme null. Det er toleransen som angir hvor nærme vi ønsker at vi skal være. Hvis vi ikke bruker en toleranseparameter, konvergerer metoden sjeldent.

d)¶

Vi kan bruke en tellevariabel i tilleg til toleransen. Da angir vi maks antall iterasjoner som metoden skal utføre. Funksjonen bør derimot gi beskjed til brukeren dersom maks antall iterasjoner er nådd uten at toleransen er nådd.

Oppgave 16¶

Arealet under kurvene kan regnes ut ved bestemt integrasjon. Siden vi har å gjøre med datapunkter og ikke funksjoner, må vi bruke numerisk integrasjon. Da tilnærmer vi arealet under kurvene med arealet til et visst antall \(n\) geometriske figurer. Det enkleste er å ta utgangspunkt i rektangler, men vi får ofte bedre resultater ved å bruke trapeser eller tilnærme toppen av de geometriske figurene med for eksempel en andregradsfunksjon (Simpsons metode). En enkel og relativt nøyaktig metode, er å benytte rektangler, der høyden i rektanglene beregnes ut fra midtpunktet på rektangelet.

Oppgave 17¶

a)¶

Den numeriske tilnærmingen er slik basert på en framoverdifferanse av funksjonsverdier:

\[f'(x) \approx \frac{f(x+dx)-f(x)}{dx}\]

Her tilnærmer vi grenseverdien i den analytisk deriverte med en endelig differanse \(dx\).

b)¶

Dersom vi deriverer funksjonen, får vi den momentane endringen i reaksjonen. I toppunktet er det ingen endring, som betyr at \(f'(x) = 0\). Dersom vi setter den deriverte lik 0, kan vi altså finne x-verdien til toppunktet \(x_{topp}\), som vi igjen kan regne ut den potensielle energien med ved å sette \(f(x_{topp})\).

c)¶

Vi kan tilnærme y-verdidifferansen med differansen mellom neste punkt og nåværende punkt (f(x+dx) - f(x)). Dette er framoverdifferansen. Hvis vi isteden tar differansen mellom nåværende punkt og forrige punkt (f(x) - f(x-dx)), får vi bakoverdifferansen. Tar vi snittet av disse to metodene, får vi midtpunktsdifferansen: (f(x + dx) - f(x - dx))/2. Midtpunktstilnærmingen gir den beste tilnærmingen til den numerisk deriverte av disse metodene.

import numpy as np
import matplotlib.pyplot as plt

# Konstanter og variabler
k = 7.30E-4          # Ratekonstanten (s^-1)
c0_peroksid = 1.0    # mol/L
c0_oksygengass = 0   # mol/L
c0_vann = 0          # mol/L

# Tidssteg
dt = 1E-2  # s
tid = 3600 # s
N = int(tid/dt) # Antall intervaller

# Arrayer og initialisering
c_H2O2 = np.zeros(N+1)
c_O2 = np.zeros(N+1)
c_H2O = np.zeros(N+1)
t = np.zeros(N+1)

c_H2O2[0] = c0_peroksid
c_O2[0] = c0_oksygengass
c_H2O[0] = c0_vann

# Eulers metode
for i in range(N):
    cder_H2O2 = -k*c_H2O2[i]
    cder_O2 = -0.5*cder_H2O2
    cder_H2O = -cder_H2O2
    c_H2O2[i+1] = c_H2O2[i] + cder_H2O2*dt
    c_O2[i+1] = c_O2[i] + cder_O2*dt
    c_H2O[i+1] = c_H2O[i] + cder_H2O*dt
    t[i+1] = t[i] + dt

plt.plot(t,c_H2O2,label='$H_2O_2$')
plt.plot(t,c_O2,label='$O_2$')
plt.plot(t,c_H2O,label='$H_2O$')
plt.xlabel('Tid (s)')
plt.ylabel('Konsentrasjon (mol/L)')
plt.legend()
plt.show()

# Definerer funksjon for å beregne det bestemte integralet med n rektangler mellom a og b vha. midtpunktstilnærmingen.
def rektangelmetoden_midt(f, a, b, n):
# Definerer arealet
    A = 0.0        
# Regner ut bredden av hvert rektangel
    h = (b-a)/n 
# For hvert rektangel...
    for k in range (n):    
    # beregn høyden til rektangelet som høyden i midtpunktet til rektangelet
        A = A + f(a + (1+2*k)*(h/2))
    # Beregn til slutt arealet ved å gange med bredden av hvert rektangel
    return A*h

Oppgave 21¶

I linja “y[i+1] = y[i] + yder(x, y)*dx” må \(x\) og \(y\) indekseres siden vi her mater inn hele arrayer i funksjonen. Vi er bare interessert i verdi \(i\): “…+ yder(x[i], y[i])”.

x[i+1] + = dx innebærer at vi tar x[i+1] = x[i+1] + dx. Vi er her ute etter den forrige verdier + dx, altså x[i] + dx.

Når vi kaller på funksjonen integrate_ODE, må vi huske på å lagre verdiene i variabler. Nå lager vi verdiene, men vi lagrer dem ingen steder. Vi kan f.eks. skrive “x, y = integrate_ODE(dy_dt,0,4,1,1E-5)”.

Vi kan heller ikke sette y0 = y, fordi y er en array. Parameteren y i funksjonen burde derfor vært y0.
